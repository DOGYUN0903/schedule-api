# schedule_api

## 프로젝트 개요

프로젝트명: 일정 관리 API 구현 Ver2.

본 API는 회원 가입을 기반으로 일정을 등록하고, 조회, 수정, 삭제할 수 있는 기능을 제공합니다.

## Git 컨벤션
| 태그       | 설명                                       |
|------------|--------------------------------------------|
| `feat`     | 새로운 기능 추가                           |
| `fix`      | 버그 수정                                  |
| `refactor` | 코드 리팩토링 (기능 변화 없이 구조 개선)   |
| `docs`     | 문서 수정 (예: README, 주석 등)            |
| `chore`    | 기타 설정, 빌드, 패키지 등                 |
